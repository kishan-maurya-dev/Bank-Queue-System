# Bank-Queue-System
The Bank Queue System is a Java program that simulates bank customer management using the Queue data structure. It follows the FIFO (First In First Out) principle to serve customers in order. The system allows adding customers, serving them, and displaying the queue, demonstrating a real-world application of queues.
# Bank Queue System

## 📌 Project Overview
The Bank Queue System is a simple Java application that simulates how customers are managed in a bank using the Queue data structure.

A queue follows the FIFO (First In First Out) principle, meaning the first customer to enter the queue is the first one to be served.

This project demonstrates the practical use of queues in real-life applications such as bank customer management.

---

## 🛠 Technologies Used
- Java
- Queue Data Structure
- Object Oriented Programming

---

## ⚙️ Features
• Add customer to queue (Enqueue)  
• Serve customer (Dequeue)  
• Display current queue  
• Check if queue is empty  
• Check if queue is full  

---

## 🧠 Data Structure Used
Queue (Array Implementation)

Operations used:
- Enqueue
- Dequeue
- Display Queue
